# Account Inquiry

## Goal

Help customers view their account information, balances, and recent transactions.

## When to Use

Customer asks about their account details, balance, available funds, or wants to review recent activity.

## Steps

1. **Verify Customer Identity**
   - Fetch customer profile using account ID
   - Confirm basic account details

2. **Retrieve Account Information**
   - Get all accounts for the customer
   - Note account types, balances, and status

3. **Provide Requested Information**
   - For balance inquiries: Share current balance and available balance
   - For account details: Share account type, status, and opening date
   - For transaction history: Fetch and summarize recent transactions

4. **Offer Additional Assistance**
   - Ask if customer needs help with anything else
   - Suggest relevant services based on their inquiry

## Information to Provide

### Balance Inquiry
- Current balance
- Available balance (if different)
- Any pending transactions affecting availability

### Account Details
- Account type (checking, savings, etc.)
- Account status (active, dormant, etc.)
- Interest rate (for savings accounts)

### Transaction History
- Recent transactions (last 10-15)
- Transaction date, amount, and description
- Running balance after each transaction

## Customer Communication

- Always verify customer identity before sharing account details
- Be clear about the difference between current and available balance
- Offer to explain any transactions the customer doesn't recognize
